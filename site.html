!qadd

Adds a quote to the list of quotes.

Example: !qadd I like adding quotes.

!randomnumber

Generates a random number.

Example: !randomnumber

!coords

Displays the bot's current coordinates.

Example: !coords

!help

Displays the list of available commands.

Example: !help

!kill

Kills the bot.

Example: !kill

!serverinfo

Provides information about the server.

Example: !serverinfo

!msgs

Displays the number of messages sent by a user.

Example: !msgs cyclicalBot

!botinfo

Displays information about the bot.

Example: !botinfo

!report

Reports a player with a specified reason.

Example: !report cyclicalBot griefed my base.

!clout

Displays the number of times a user has been mentioned.

Example: !clout cyclicalBot

!discord

Provides the link to the Discord server.

Example: !discord

!faq

Displays a random FAQ.

Example: !faq

!addfaq

Adds a fact to the FAQ list.

Example: !addfaq I like adding facts.

!joindate

Displays the join date of a user.

Example: !jd cyclicalBot or !joindate cyclicalBot

!pokemon

Displays a random Pokémon or a specified Pokémon by number.

Example: !pokemon or !pokemon 150

!nword

Displays the number of times a user has said the n-word.

Example: !nword username or !nword top

!joke

Tells a random joke.

Example: !joke

!ping

Displays the users ping.

Example: !ping

!quote

Displays a random quote.

Example: !quote

!magic8ball

Will provide an answer to your question.

Example: !magic8ball Do my parents love me? or !m8b Do my parents love me?

!bestping

Displays the user with the lowest ping.

Example: !bestping or !bp

!playtime

Displays the playtime of a user.

Example: !playtime cyclicalBot or !pt cyclicalBot

!worsting

Displays the user with the highest ping.

Example: !worstping or !wp

!uptime

Displays how long the bot has been online.

Example: !uptime

!coinflip

Flips coin and displays the outcome in chat.

Example: !coinflip
